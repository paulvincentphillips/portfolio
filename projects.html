pumpspy.ie

Full stack fuel comparison site developed in my spare time. Users may sign up to edit and update prices. This is required as no API for fuel prices exists in Ireland. React.js front end, backend with Node.js & Express & PostgreSQL database. Hosted on VPS running Debian. PM2 used in production to serve backend & NGINX used to serve static resources. Several hundred daily active users.
